## Training Types to Track

- **FERPA Initial** - ED.gov online modules (required before any data access)
- **FERPA Annual Refresher** - Annual recertification via ED.gov
- **Security Awareness** - Phishing, password security, etc.
- **Incident Response** - How to report and respond to incidents
- **Role-Specific** - Additional training for specific roles

## Key Dates to Monitor

1. **Next Training Due** - Set reminders 30 days before
2. **Background Check Date** - Verify before Tier 2+ access
3. **NDA Signed Date** - Must be before any access granted
4. **Certificate Expiration** - Some modules may have expiration dates

## Compliance Checks

### Monthly Review
- [ ] Check for training expiring in next 60 days
- [ ] Send reminders to employees with upcoming deadlines
- [ ] Verify all new employees completed initial training

### Quarterly Review
- [ ] Audit access tiers against current roles
- [ ] Verify all active employees have current training
- [ ] Review and archive records for departed employees

### Annual Review
- [ ] Verify all employees completed annual refresher
- [ ] Update training requirements based on policy changes
- [ ] Archive previous year's records

## Retention Policy

- **Active Employees:** Maintain current and historical records
- **Departed Employees:** Retain for 3 years after departure
- **Certificates:** Store indefinitely in compliance folder
- **Training Tracker:** Backup monthly, retain permanently

## Audit Readiness

This tracker should be available to demonstrate:
- ✅ All employees with data access have completed FERPA training
- ✅ Background checks completed before access granted
- ✅ Annual refresher training is current
- ✅ Training certificates are on file
- ✅ Access tiers match job roles and training levels

## Storage Location

**Store in secure compliance folder OUTSIDE of git repository:**
- Google Drive/Workspace with restricted access
- Encrypted local storage with regular backups
- Access limited to HR and Security team only

**DO NOT store in code repository or public locations.**
